## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install elr-scss-alerts --save
yarn add elr-scss-alerts
```

## Implementation

### Scss

```scss
@import "elr-scss-alerts";

.alert {
  @include elr-alert;
}
```

```scss
@import "elr-scss-alerts";

.alert-info {
  @include elr-alert-context(
    $config: (
      type: info,
    )
  );
}
```
